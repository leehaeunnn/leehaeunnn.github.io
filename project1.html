프로젝트 1: 반응형 웹 포트폴리오

프로젝트 개요

이 프로젝트는 반응형 웹 디자인 기술을 활용하여 다양한 기기에서 최적화된 사용자 경험을 제공하는 개인 포트폴리오 웹사이트를 개발한 것입니다.

사용된 기술

HTML5 CSS3 JavaScript Responsive Design CSS Grid Flexbox

주요 기능

  • 모바일 우선 접근 방식을 사용한 반응형 레이아웃
  • CSS Grid 및 Flexbox를 활용한 모던 레이아웃
  • 다크 모드 / 라이트 모드 전환 기능
  • 프로젝트 갤러리 및 모달 창
  • 접근성 고려한 디자인

개발 과정

이 프로젝트는 사용자 중심 디자인 원칙을 따라 개발되었으며, 모바일 우선 접근 방식을 적용하였습니다. 초기 와이어프레임 설계부터 시작하여 프로토타입을 만들고, 사용자 피드백을 반영하여 지속적으로 개선했습니다.

배운 점

이 프로젝트를 통해 반응형 웹 디자인의 중요성과 다양한 화면 크기에 적응하는 레이아웃을 설계하는 방법을 배웠습니다. 또한 사용자 경험을 최우선으로 고려하는 디자인 사고 방식을 발전시킬 수 있었습니다.
