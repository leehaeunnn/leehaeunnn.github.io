수학 채점 시스템

Next.js와 MongoDB를 활용한 온라인 수학 문제 채점 시스템입니다. 교사들이 문제를 등록하고 학생들의 답안을 효율적으로 채점할 수 있도록 설계되었습니다.

Next.js 14 React 18 TypeScript MongoDB Mongoose Tailwind CSS MathJax React LaTeX

주요 기능

  • 수학 문제 생성 및 관리
  • LaTeX 수식 렌더링 지원
  • 학생 답안 제출 시스템
  • 자동 채점 기능
  • 상세한 채점 결과 피드백
  • 문제별 통계 및 분석
  • 반응형 웹 디자인
  • 실시간 데이터 업데이트

프로젝트 구조

이 프로젝트는 다음과 같은 구조로 구성되어 있습니다:

  • /app - Next.js 앱 라우터 및 페이지 컴포넌트
  • /models - MongoDB 스키마 정의 (Problem, Submission)
  • /lib - 데이터베이스 연결 및 유틸리티 함수
  • /api - RESTful API 엔드포인트
참고: 이 프로젝트는 MongoDB 데이터베이스 연결이 필요합니다. 로컬 환경에서 실행하려면 MongoDB 설정과 환경 변수 구성이 필요합니다.
