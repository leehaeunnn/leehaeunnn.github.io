2차원 포물선 운동 비디오 분석기

비디오에서 물체의 2차원 포물선 운동을 추적하고 분석하는 Python GUI 애플리케이션입니다. 물리 교육 및 실험 데이터 분석에 활용할 수 있도록 개발되었습니다.

Python 3 OpenCV NumPy Matplotlib Tkinter PIL

주요 기능

  • 비디오 파일 불러오기 및 프레임 단위 분석
  • 수동/자동 물체 추적 모드
  • 실제 거리 스케일 보정 (calibration)
  • 좌표계 원점 설정 기능
  • 위치, 속도, 가속도 데이터 실시간 계산
  • 포물선 궤적 시각화 및 그래프 생성
  • 데이터 JSON 형식 저장/불러오기
  • 한글 인터페이스 지원

작동 원리

1. 비디오 로드
실험 비디오 파일 선택
2. 스케일 보정
실제 거리 기준 설정
3. 물체 추적
프레임별 위치 마킹
4. 데이터 분석
물리량 계산 및 시각화

실행 방법

# 필요한 패키지 설치 pip install -r requirements.txt # 프로그램 실행 python video_analyzer.py
참고: 이 프로그램은 Python 3.7 이상과 시스템에 설치된 OpenCV 라이브러리가 필요합니다. Windows 환경에서는 Visual C++ Redistributable이 필요할 수 있습니다.
