AI 기반 수학 채점 시스템

Google Gemini AI를 활용한 고등학교 수학 문제 자동 채점 솔루션

작동 과정

1

이미지 업로드

문제지, 정답지, 학생 답안을 PNG/JPG 형식으로 업로드합니다.

2

AI 분석

Google Gemini 2.5 Pro가 이미지를 분석하여 문제와 답안을 이해합니다.

3

자동 채점

미리 설정된 채점 기준에 따라 부분 점수까지 세밀하게 채점합니다.

4

결과 출력

채점 결과와 피드백을 CSV 파일로 내보내어 관리합니다.

Gemini 2.5 Pro

Google의 최신 AI 모델을 사용하여 수학 문제를 정확하게 이해하고 채점합니다.

이미지 인식

손글씨, 수식, 그래프 등 다양한 형태의 답안을 인식할 수 있습니다.

부분 점수

풀이 과정을 분석하여 단계별 부분 점수를 부여합니다.

상세한 피드백

틀린 부분에 대한 구체적인 피드백을 제공합니다.

CSV 내보내기

채점 결과를 엑셀에서 바로 열 수 있는 CSV 형식으로 저장합니다.

실시간 진행 상황

채점 진행 상황을 실시간으로 확인할 수 있습니다.

API 사용 정보

Google Gemini API를 사용합니다.

  • API 키가 필요합니다 (Google AI Studio에서 발급)
  • 무료 할당량 내에서 사용 가능합니다
  • 환경 변수(.env)로 안전하게 관리됩니다
  • API 키는 절대 GitHub에 업로드하지 마세요!

기술 스택

Python 3 Flask Gemini 2.5 Pro API Google AI SDK Pillow (PIL) Pandas HTML/CSS/JavaScript 
# 설치 및 실행
pip install -r requirements.txt

# .env 파일에 API 키 설정 (절대 GitHub에 업로드 금지!)
GEMINI_API_KEY=your-api-key-here  # Google AI Studio에서 발급
GEMINI_MODEL=gemini-2.5-pro-latest

# 서버 실행
python main.py
